How To Develop Documentation for a Project

As a Design for Communication and Experience (DCX) student in Media, Arts, and Design, documentation for projects is an extremely familiar topic for me. One way to go about it would be taking as many photos or screenshots of every step that we do, for future reference for others and ourselves. Notes and research regarding our projects are then added on to accompany these photos to add more clarity, and most importantly, be as detailed as possible!


How to Use Git to Maintain the Site

We can use GitHub to maintain our website by simply uploading all our files into GitHub, once we have our repository, and enabled GitHub Pages. Our website would take a while to load once all the files have been added, and can be found by selecting the “main” branch as the source. Any extra changes made can also be simply updated within the already added files, and adding new ones.
