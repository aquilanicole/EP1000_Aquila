Assignment 3

Task 1: a. Connect a LED with a current limiting resistor (220 ohm) to port 6 of the Arduino Uno board b. Write a program to blink the LED in a variety of patterns, i.e Blink 1 time, delay (Milliseconds) then Blink 2 time etc
Checkout Task 1 LED Blink here!
Task 2: a. Using the same technique, connect 3 LEDs of different colours to the Arduino board using pins 6, 7, and 8 b. Write a program to produce running lights i.e. the LEDs are lit in sequence e.g. R -> G -> B -> R -> G -> B and so on.
Checkout Task 2 RGB light sequence here!
Task 3: a. Add a switch to the circuit in task 1. (You should use a pull-up resistor of 10K if you are operating in INPUT mode.) b. Write a code when the switch is pressed, the LEDs will light up. Pressing the switch again will turn the LEDs off.
Checkout Task 3 Switch here!
Task 4: a. Start with LEDs are all in the OFF state. b. Pressing the switch, turns on only the RED LED. c. Pressing the switch again, turns on only the GREEN LED d. Pressing the switch again, turns on only the BLUE LED e. Pressing the switch again, turns on all LEDs f. Pressing the switch again, turns off all LEDs returning to the first state
